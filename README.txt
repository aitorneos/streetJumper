														STREET JUMPER :

USAGE INSTRUCCIONS :

1 . Install AndEngine - AnchorCenter in Eclipse : https://github.com/nicolasgramlich/AndEngine/tree/GLES2-AnchorCenter
	
Extensions :
	
Augmented Reality Extension : https://github.com/nicolasgramlich/AndEngineAugme	ntedRealityExtension.git
	
Multiplayer Extension : https://github.com/nicolasgramlich/AndEngineMultip	layerExtension.git
	
Physics Extension : https://github.com/nicolasgramlich/AndEnginePhysic	sBox2DExtension.git
	
Examples : https://github.com/nicolasgramlich/AndEngineExamples.git
	
				
2. Java Compiler : 1.6
	
3. Android Version : >= 2.2 && < 3.0
	
4. Run project as Android Project in Eclipse 
	
5. VERY IMPORTANT : This Game uses AndEngine graphic libraries GLES2 Anchor Center version (openGL 2.0) , and in consequence, in devices which Android version is bigger than 2.3.6 
or openGL version is a little diferent could produce some game failures (physics, animations, collisions, etc). To run this game without surprises is very advisable to run it in a
device with android version between 2.2 and 3.0 with openGL 2.0 (GLES2).
